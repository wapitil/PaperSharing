# 更新日志

## 2026-06-17

### 新增

- 新增独立阅读端 `reader.html` 和 `reader.js`。
- 新增独立管理端 `admin.html` 和 `admin.js`。
- 阅读端支持从本地存储读取分享端保存的阅读经验。
- 阅读端支持按论文聚合阅读经验。
- 阅读端支持论文搜索，可搜索标题、作者、年份和链接。
- 阅读端支持载入示例数据。
- 管理端支持查看全部本地记录。
- 管理端支持删除单条经验。
- 管理端支持清空全部本地数据。
- 管理端支持导出和导入 JSON。
- 阅读端保持只读，删除和清空操作后续移到管理端。

### 调整

- 分享端顶部入口从“查看阅读界面”调整为“查看阅读库”。
- 阅读端顶部入口从“去分享”调整为“添加阅读经验”。
- 阅读端仅保留“按分享者查看”的阅读方式。
- 阅读端论文列表按最近上传时间排序。
- 同一论文下的分享者经验按上传时间倒序展示。
- 分享者名称恢复为手动填写，暂不使用账户绑定。
- 顶部同级操作统一为相同字体、字号和按钮高度。

### 校验

- 分享端保存时新增内容校验：至少需要填写“为什么选择这一篇文章”、一个阅读点或“我的看法”之一。

### 说明

- 当前数据仍保存在浏览器 `localStorage` 中。
- 当前版本用于验证分享端和阅读端的最小可用流程，暂未接入后端和账户系统。
